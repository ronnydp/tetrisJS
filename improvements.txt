Mejoras y adiciones que podrías implementar:

1. Funcionalidades del juego:
    •  Pausa del juego: Tecla para pausar/reanudar (Espaciebar o P)
    •  Reinicio: Tecla R para reiniciar sin alert
    •  Niveles de dificultad: Aumentar velocidad según puntuación ✅
    •  Preview de siguiente pieza: Mostrar la próxima pieza
    •  Hold/Reserva: Guardar una pieza para usar después
    •  Líneas eliminadas: Contador de líneas completadas ✅
    •  Efectos de línea completa: Animación cuando se elimina una línea

2. Mejoras visuales:
    •  Colores por pieza: Cada tipo de pieza con su color característico
    •  Grid del tablero: Líneas para delimitar las celdas
    •  Animaciones: Efectos al eliminar líneas, caída más suave
    •  Partículas: Efectos visuales al completar líneas
    •  Mejor UI: Panel de información más atractivo

3. Audio:
    •  Música de fondo: Tema clásico de Tetris
    •  Efectos de sonido: Para rotación, líneas completadas, game over
    •  Control de volumen: Botones para silenciar/ajustar

4. Mejoras técnicas:
    •  Mejor sistema de colores: Array de colores para cada pieza
    •  Optimización de renderizado: Solo dibujar cuando hay cambios
    •  Sistema de estados: Menú, jugando, pausado, game over
    •  Guardado de puntuación: LocalStorage para high scores
    •  Responsive design: Adaptar a diferentes tamaños de pantalla

5. Controles adicionales:
    •  Caída rápida: Mantener ↓ para caída más rápida
    •  Hard drop: Tecla para caída instantánea (Spacebar)
    •  Rotación inversa: Ctrl+↑ para rotar al revés

6. Mejoras en el código:
    •  Constantes para colores: Definir paleta de colores
    •  Función de reset: Separar la lógica de reinicio
    •  Mejor manejo de game over: Estado en lugar de alert
    •  Documentación: Comentarios más detallados
